Usage

After you unzip or git clone the repository, run test.py directory in the top directory.

This project requires python2.7, python3.6 and cmake

$ python3.6 test.py

As previously mentioned, 7 map choices will be provided. The program will automatically run through each algorithm one by one, and output the running time for each algorithm in your console.

Our steps of using this program and finished this project was -

  1. Made a google sheet, listing our testing platforms one by one.
  2. Listed the 6 pre-built maps under those platforms.
  3. Listed the 7 alforithms under those maps.
  4. Run the program on each machine, and test each of the 6 pre-built maps one by one, and record the running time on the sheet.
  5. Made bar graphs on the google sheet to get a clear idea of the comparison results.
